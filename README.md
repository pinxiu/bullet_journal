# Bullet Journal

## Install

```
pip3 install bullet_journal
```

## Usage

### Add to-do task
```
bj add [date]
date: optional argument in the format of yyyy_mm_dd
      default is today
```

### Close to-do task
```
bj close [date]
date: optional argument in the format of yyyy_mm_dd
      default is today
```

### Migrate to-do task to a new date
```
bj migrate [date]
date: optional argument in the format of yyyy_mm_dd
      this is the date where the task currently belongs to
      default is today
```
